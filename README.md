# ☕ Coffee Shop Sales Analysis

## 📌 Project Overview
This project focuses on analyzing coffee shop sales data to generate meaningful business insights.  
The analysis is performed using **MySQL for data processing** and **Power BI for visualization**.

The goal is to understand sales performance, customer behavior, and identify trends to support data-driven decision-making.

---

## 🎯 Objectives
- Analyze overall sales performance
- Track key KPIs like Total Sales, Orders, and Quantity Sold
- Identify Month-over-Month (MoM) growth trends
- Understand customer purchasing patterns
- Analyze product and store performance
- Identify peak sales days and hours

---

## 🛠️ Tools & Technologies Used
- **MySQL** – Data cleaning, transformation, and analysis  
- **Power BI** – Dashboard creation and visualization  
- **SQL** – Complex queries, aggregations, and window functions  

---

## 📂 Dataset Description
The dataset contains transactional data with the following fields:
- Transaction ID  
- Transaction Date & Time  
- Product Category & Type  
- Quantity Sold  
- Unit Price  
- Store Location  

---

## 🔄 Data Cleaning & Preprocessing
- Converted `transaction_date` to DATE format  
- Converted `transaction_time` to TIME format  
- Fixed column naming issues  
- Ensured proper data types for accurate analysis  

---

## 📊 Key KPIs
- **Total Sales**
- **Total Orders**
- **Total Quantity Sold**
- **Month-over-Month Growth**

---

## 📈 Analysis Performed
- Monthly Sales Trends  
- Daily Sales Analysis  
- Weekday vs Weekend Sales  
- Hourly Sales Patterns  
- Top Products by Sales  
- Sales by Product Category  
- Sales by Store Location  

---

## 📊 Dashboard Features
The Power BI dashboard includes:
- KPI Cards (Sales, Orders, Quantity)
- Sales Trend Over Time
- Product Category Analysis
- Top 10 Products
- Store Location Performance
- Day & Hour Heatmap
- Calendar View for Daily Sales

## 💡 Key Insights

- Found best-selling products and categories  
- Observed higher sales during specific hours of the day

- Compared weekday vs weekend performance
